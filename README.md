# radd-learn-git
Shared Repo for RADD #github channel to help participants learn git

## What to do?

* [Create a GitHub account](https://help.github.com/en/github/getting-started-with-github/signing-up-for-github)
* [Setup GIT on your local machine](https://help.github.com/en/github/getting-started-with-github/set-up-git)
* [Fork this repository](https://help.github.com/en/github/getting-started-with-github/fork-a-repo)
* [Clone your fork to your local machine](https://help.github.com/en/github/creating-cloning-and-archiving-repositories/cloning-a-repository)
* [Create a branch](https://git-scm.com/book/en/v2/Git-Branching-Basic-Branching-and-Merging)
* [Make changes to README.md](https://help.github.com/en/github/writing-on-github/about-writing-and-formatting-on-github)
* [Commit your changes](https://git-scm.com/book/en/v2/Git-Basics-Recording-Changes-to-the-Repository)
* [Push your changes up to your github repository](https://help.github.com/en/github/using-git/pushing-commits-to-a-remote-repository)
* [Create a pull request](https://help.github.com/en/github/collaborating-with-issues-and-pull-requests/about-pull-requests)
